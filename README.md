## Installation

1. Download the latest release.
2. Extract the downloaded archive.
3. Place the included mod folder into your local **r2modman** profile (usually) at:
```txt  
C:/Users/YOUR_USER/AppData/Roaming/r2modmanPlus-local/RiskOfRain2/profiles/YOUR_PROFILE/BepInEx/plugins/
```
4. Run the game once so the config file is generated.
5. Open the generated config file and edit the values however you like.

---
## Usage

After first launch, SurvivorTuner will generate its config file. From there, you can adjust available values to tune Commando's stats and abilities to your preference.

This mod is intended to give players a simple way to experiment with survivor tuning, whether for balance, challenge, personal preference, or just messing around with custom values.
